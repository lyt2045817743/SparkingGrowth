# what_to_study_today
## 简介：以一种随机的方式生成自己今天要学的内容。

## 功能模块
[x] 随机生成章节目录
[x] 学习记录展示
[ ] 在线文档笔记模块
[ ] 日历用于可视化学习成果。分为大小日历视图，包含：每日学习次数、章节名称、【单次学习时长】
