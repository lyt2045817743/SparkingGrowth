# what_to_study_today
## 简介：以一种随机的方式生成自己今天要学的内容。

## 功能模块
[x] 随机生成章节目录
[x] 学习记录展示
[ ] 在线文档笔记模块
[ ] 日历用于可视化学习成果。分为大小日历视图，包含：每日学习次数、章节名称、【单次学习时长】
[ ] 复习功能。每天学习前需要复习前一天的笔记，才能使用随机生成章节功能
[ ] 设置学习时长增加紧迫感，设置后即开始学习。每次只统计从开始到暂停/设置的时长结束的时间，也支持手动更改
